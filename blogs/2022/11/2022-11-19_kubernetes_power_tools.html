Kubernetes Power Tools

Managing increasing number of kubernetes clusters can become cumbersome over time and there are few power tools available to make your life easy. In this blog, we will discuss few power tools which you can use to reduce the repeatibility and increase your efficieny.


Kubectx

When you have switch context between different k8 clusters multiple times, typing the kubectl command can be time consuming and kubectx comes to the rescue. With kubectx, you can easily switch between different k8 clusters with easy to remember aliases.

Installation
On Linux:

sudo apt install kubectx
On Windows:
choco install kubectx
More details on installation at kubectx

To list all the contexts saved in the kubeconfig file:
kubectx

To switch the context to a cluster 'prod-k8':
kubectx prod-k8
